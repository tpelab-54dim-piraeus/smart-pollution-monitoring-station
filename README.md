# smart-pollution-monitoring-station
## Περιγραφή

Ο Σταθμός Παρακολούθησης Περιβαλλοντικής Ρύπανσης επικεντρώνεται στη διερεύνηση διαφορετικών μορφών μόλυνσης που επηρεάζουν την ποιότητα ζωής στο σχολείο και την ευρύτερη κοινότητα. Οι μαθητές χρησιμοποίησαν Arduino και Mind+ για να προγραμματίσουν αισθητήρες ποιότητας αέρα τύπου MQ, αισθητήρα ήχου για την καταγραφή της ηχορύπανσης και αισθητήρα φωτεινότητας για τη μελέτη της φωτορύπανσης. Οι μετρήσεις δεν αντιμετωπίζονται ως απόλυτες τιμές, αλλά κατηγοριοποιούνται παιδαγωγικά σε επίπεδα χαμηλής, μέτριας και υψηλής ρύπανσης, με οπτικές και ηχητικές ειδοποιήσεις όταν ξεπερνιούνται προκαθορισμένα όρια. Μέσα από πειραματισμούς και παρατηρήσεις, οι μαθητές κατανοούν πώς η ανθρώπινη δραστηριότητα επηρεάζει το περιβάλλον, αναπτύσσουν περιβαλλοντική και κοινωνική συνείδηση και προβληματίζονται πάνω σε τρόπους περιορισμού της ρύπανσης. Το έργο ενισχύει την κριτική σκέψη, την υπευθυνότητα και τη συνεργασία, δίνοντας στους μαθητές ενεργό ρόλο στην προστασία του περιβάλλοντος.

## Στόχοι
Οι μαθητές θα μπορούν:

**Γνωστικοί στόχοι**
 - να αναγνωρίζουν διαφορετικές μορφές ρύπανσης (ατμοσφαιρική, ηχορύπανση, φωτορύπανση)
 - να κατανοούν πώς η ανθρώπινη δραστηριότητα επηρεάζει το περιβάλλον
 - να εξηγούν βασικές έννοιες ποιότητας αέρα και περιβαλλοντικού κινδύνου

**Στόχοι δεξιοτήτων (STEM & Προγραμματισμός)**
 - να προγραμματίζουν αισθητήρες με το οπτικό περιβάλλον Mind+
 - να χρησιμοποιούν μεταβλητές, συνθήκες και επαναλήψεις
 - να κατηγοριοποιούν τιμές σε επίπεδα (χαμηλή–μέτρια–υψηλή ρύπανση)
 - να δημιουργούν συστήματα ειδοποίησης (LED, buzzer)

**Στάσεις – αξίες**
 - να αναπτύξουν οικολογική και κοινωνική συνείδηση
 - να καλλιεργήσουν υπεύθυνη συμπεριφορά στο δημόσιο χώρο
 - να ενισχύσουν την κριτική σκέψη και να προβληματιστούν
 - να συνεργαστούν δημιουργικά και με υπευθυνότητα μέσα στην ομάδα
